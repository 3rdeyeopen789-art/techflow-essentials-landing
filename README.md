# TechFlow Essentials Landing Page

Lead magnet landing page for "The Agent Playbook" - Free AI Prompt Engineering Guide.

## Features

- 🎨 Modern, conversion-focused design
- 📧 Klaviyo email capture integration
- ⚡ Fast static site built with React + Vite
- 📱 Fully responsive
- 🎯 Optimized for Facebook ad traffic

## Tech Stack

- React 19
- Vite
- TypeScript
- Tailwind CSS 4
- Klaviyo Forms

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```

## Deployment

This site is deployed on Vercel. Any push to the `main` branch will automatically trigger a new deployment.

### Environment Variables

No environment variables needed - Klaviyo form is embedded directly.

## Email Capture

Email capture is handled by Klaviyo's native embedded form (Form ID: UMUE4m).
Emails are automatically added to the "Email List" (List ID: VYimX4).
